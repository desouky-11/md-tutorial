ملف **Markdown مطوّل وشامل** مع **أمثلة عملية لكل ميزة**، بحيث يكون مرجع كامل.

---

````markdown
# مرجع شامل للـ Markdown مع أمثلة عملية

---

## 1. العناوين (Headers)

```markdown
# العنوان الرئيسي (H1)
## العنوان الفرعي (H2)
### العنوان الثالث (H3)
#### العنوان الرابع (H4)
##### العنوان الخامس (H5)
###### العنوان السادس (H6)
````

**نتيجة العرض:**

# العنوان الرئيسي (H1)

## العنوان الفرعي (H2)

### العنوان الثالث (H3)

#### العنوان الرابع (H4)

##### العنوان الخامس (H5)

###### العنوان السادس (H6)

---

## 2. النصوص (Text)

```markdown
*نص مائل* أو _نص مائل_
**نص عريض** أو __نص عريض__
***نص مائل وعريض***  
~~نص مشطوب~~
```

**نتيجة العرض:**

*نص مائل* أو *نص مائل*
**نص عريض** أو **نص عريض**
***نص مائل وعريض***
~~نص مشطوب~~

---

## 3. القوائم (Lists)

### 3.1 قائمة غير مرتبة

```markdown
- تفاحة
- موز
- كيوي
```

**نتيجة العرض:**

* تفاحة
* موز
* كيوي

### 3.2 قائمة مرتبة

```markdown
1. تعلم Markdown
2. كتابة مستندات
3. مشاركة الملفات
```

**نتيجة العرض:**

1. تعلم Markdown
2. كتابة مستندات
3. مشاركة الملفات

### 3.3 قوائم منطقية (Task Lists)

```markdown
- [x] تم
- [ ] لم يتم
- [ ] جاري العمل
```

**نتيجة العرض:**

* [x] تم
* [ ] لم يتم
* [ ] جاري العمل

---

## 4. الروابط والصور

### 4.1 الروابط

```markdown
[Google](https://www.google.com)
```

**نتيجة العرض:**

[Google](https://www.google.com)

### 4.2 الصور

```markdown
![مثال صورة](https://picsum.photos/id/0/200/300)
```

**نتيجة العرض:**

![![مثال صورة](https://picsum.photos/id/0/200/300)


---

## 5. الاقتباسات (Blockquotes)

```markdown
> هذا اقتباس
> يمكن أن يمتد على عدة أسطر
```

**نتيجة العرض:**

> هذا اقتباس
> يمكن أن يمتد على عدة أسطر

---

## 6. الكود (Code)

### 6.1 كود داخل السطر

```markdown
استخدم `console.log()` للطباعة في الجافاسكريبت
```

**نتيجة العرض:**

استخدم `console.log()` للطباعة في الجافاسكريبت

### 6.2 كود متعدد الأسطر (Code Block)

````markdown
```python
def greet(name):
    print(f"مرحبا {name}")
greet("أحمد")
````

````

**نتيجة العرض:**
```python
def greet(name):
    print(f"مرحبا {name}")
greet("أحمد")
````

**نتيجة العرض:**

```markdown
| ليلى    | 30   | دبي         |
| محمد    | 22   | الرياض      |
```

**نتيجة العرض:**

| الاسم | العمر | المدينة |
| ----- | ----- | ------- |
| أحمد  | 25    | القاهرة |
| ليلى  | 30    | دبي     |
| محمد  | 22    | الرياض  |

```markdown
يمكن دمج **النص العريض** مع *النص المائل* و `الكود`
```

**نتيجة العرض:**

يمكن دمج **النص العريض** مع *النص المائل* و `الكود`

## 10. القوائم المتداخلة (Nested Lists)

```markdown
  - خيار
```

  * موز

  * جزر

## 11. النصوص مع المحاذاة والاقتباسات الخاصة (Extended Markdown - حسب المحرر)


**نتيجة العرض:**
